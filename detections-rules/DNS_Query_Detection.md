# DNS Query Detection

## Objective

Monitor DNS query activity generated by Microsoft Sysmon to identify suspicious or potentially malicious domain lookups.

---

## SPL Query

```spl
index=* sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=22
| stats count by QueryName Image
```

---

## Detection Logic

This detection monitors **Sysmon Event ID 22**, which records DNS queries performed by processes on the monitored endpoint. It helps identify suspicious domain lookups that may indicate malware communication, phishing activity, or command-and-control (C2) traffic.

## Expected Output

The search returns:

- DNS Query Name
- Process Image
- Number of DNS Queries

This information helps identify unusual or suspicious DNS activity.
