# Network Connection Detection

## Objective

Monitor network connection events generated by Microsoft Sysmon to identify outbound and inbound connections that may indicate suspicious or unauthorized communication.

---

## SPL Query

```spl
index=* sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=3
| stats count by Image DestinationIp DestinationPort
```

---

## Detection Logic

This detection monitors **Sysmon Event ID 3**, which records network connections initiated by processes on the monitored endpoint. It helps identify unexpected external communications, malware beaconing, or unauthorized network activity.

---

## Expected Output

The search returns:

- Process Image
- Destination IP Address
- Destination Port
- Number of Connections

This information helps identify suspicious network communication initiated by endpoint processes.
