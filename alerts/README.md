# 🚨 Splunk Alert Configuration

## Overview

This document describes the scheduled alerts configured in Splunk Enterprise to automatically detect suspicious activities within the SOC monitoring environment.

These alerts are based on custom SPL detection rules and help simulate real-world SOC monitoring and incident detection.

---

# Configured Alerts

## 1. PowerShell Execution Detection

**Objective**

Detect PowerShell execution on monitored Windows endpoints.

**Trigger Condition**

PowerShell process creation detected.

**Severity**

High

**MITRE ATT&CK**

T1059.001 – PowerShell

---

## 2. Failed Login Detection

**Objective**

Identify repeated Windows authentication failures that may indicate brute-force attacks.

**Trigger Condition**

Multiple failed login attempts detected.

**Severity**

High

**MITRE ATT&CK**

T1110 – Brute Force

---

## 3. DNS Query Detection

**Objective**

Monitor suspicious DNS requests generated by endpoint processes.

**Trigger Condition**

DNS query events detected.

**Severity**

Medium

**MITRE ATT&CK**

T1071.004 – DNS

---

## 4. Process Creation Detection

**Objective**

Monitor newly created processes using Sysmon.

**Trigger Condition**

New process execution detected.

**Severity**

Medium

**MITRE ATT&CK**

T1204 – User Execution

---

## 5. Network Connection Detection

**Objective**

Monitor outbound network connections initiated by endpoint processes.

**Trigger Condition**

Network connection event detected.

**Severity**

Medium

**MITRE ATT&CK**

T1071 – Application Layer Protocol

---

## 6. Suspicious Process Detection

**Objective**

Detect execution of commonly abused administrative tools such as PowerShell, CMD, PsExec, and WMIC.

**Trigger Condition**

Execution of monitored process detected.

**Severity**

High

**MITRE ATT&CK**

T1059 – Command and Scripting Interpreter

---

# Summary

The configured alerts provide automated monitoring for common attack techniques and enable faster detection of suspicious activities within the simulated SOC environment.
