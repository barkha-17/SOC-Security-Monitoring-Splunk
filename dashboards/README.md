# 📊 Splunk SOC Dashboard

## Overview

This folder contains the exported Splunk dashboard used in this Security Operations Center (SOC) project.

The dashboard provides real-time visibility into security events collected from Windows endpoints using Microsoft Sysmon and Windows Event Logs.

## Dashboard Features

The dashboard includes visualizations for:

- Total Security Events
- Failed Login Attempts
- Successful Logins
- Endpoint Telemetry
- DNS Query Activity
- Network Connections
- Process Creation Events
- Top Hosts
- Top Processes
- Security Event Trends

## Dashboard File

- **SOC_Dashboard.json** – Exported Splunk dashboard configuration

The dashboard can be imported into Splunk Enterprise to recreate the monitoring environment used in this project.
